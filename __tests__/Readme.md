# Configuration

## Appium
```$xslt
npm install -g appium
```

- appium-doctor: Optional
```
npm install -g appium-doctor
```

## Carthage 
```$xslt
brew install carthage
```
After installing check and add `<Carthage install folder>/bin` to PATH environment


## Start Appium server
```$xslt
appium
```
